**Отчеты** — объекты для обработки накопленной информации и получения сводных данных в удобном виде. Используют **систему компоновки данных (СКД)** или произвольные алгоритмы.

### Ключевые моменты

**СКД (Система компоновки данных)** — основной механизм построения отчетов. Позволяет создавать структурированные отчеты без программирования.

**Макет** — описание структуры отчета (поля, группировки, ресурсы, настройки).

**Схема компоновки данных** — XML-описание структуры отчета и связей с данными.

Отчеты, использующие систему компоновки данных, могут формироваться в фоновом режиме. 

### Основные компоненты отчета на СКД

| Компонент | Назначение | Где настраивается |
|---|---|---|
| **Источники данных** | Наборы данных для отчета (запросы, объекты, таблицы значений) | Схема компоновки данных → Наборы данных |
| **Поля** | Отдельные данные из источников (Счет, Сумма, Контрагент) | Схема компоновки данных → Поля |
| **Ресурсы** | Данные для агрегации (суммы, количества) | Схема компоновки данных → Ресурсы |
| **Группировки** | Иерархическая структура отчета (по организациям, периодам) | Макет → Структура отчета |
| **Отборы** | Условия фильтрации данных | Настройки отчета или схема компоновки |
| **Параметры** | Переменные, задаваемые перед выполнением (период, организация) | Схема компоновки данных → Параметры |
